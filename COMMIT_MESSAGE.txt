feat: 添加可执行程序暴露功能 (bin)

## 功能概述

实现了包可执行程序暴露功能，允许包开发者在 toolset.json 中配置 bin 字段，
管理器会在安装时自动创建符号链接到 ~/.cursortoolsets/bin/ 目录。

## 使用示例

### 包配置

```json
{
  "name": "my-toolset",
  "version": "1.0.0",
  "bin": {
    "mytool": "bin/mytool",
    "helper": "scripts/helper.sh"
  }
}
```

### 用户使用

```bash
# 安装包
cursortoolset install my-toolset

# 添加到 PATH（一次性）
export PATH="$HOME/.cursortoolsets/bin:$PATH"

# 直接使用命令
mytool --help
helper process
```
