## 📖 核心概念

### 什么是标准化打包？

一键完成工具集包的打包、校验和配置更新：

```bash
cursortoolset pack --verify
```

### 为什么需要它？

| 之前 | 现在 |
|------|------|
| 手动配置，容易出错 | ✅ 自动验证 |
| 手动排除文件 | ✅ 智能过滤 |
| 手动计算 SHA256 | ✅ 自动计算 |
| 不知道下一步 | ✅ 清晰指南 |
| 13分钟流程 | ✅ 10秒完成 |

---

## 🎯 核心命令

```bash
# 基本打包
cursortoolset pack

# 打包并自动更新 SHA256
cursortoolset pack --verify

# 查看帮助
cursortoolset pack --help
```
